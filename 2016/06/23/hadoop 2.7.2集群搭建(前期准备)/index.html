hadoop 2.7.2集群搭建(前期准备)

  • hadoop 2.7.2 前期准备

修改Linux主机名

修改/etc/sysconfig/network中的hostname

修改network的HOSTNAME项。点前面是主机名，点后面是域名。没有点就是主机名。

这个是永久修改，重启后生效。

修改/etc/hosts文件

最后，重启服务器即可

linux CentOS命令行永久修改IP地址、网关、DNS

设置IP

[root@localhost etc]# vi /etc/sysconfig/network-scripts/ifcfg-eth0

修改网关

vim /etc/sysconfig/network

增加一条网关信息

修改DNS

vim /etc/resolv.conf

增加DNS信息即可

重新加载网络配置

修改主机名和IP的映射关系

vi /etc/hosts

你的集群要用几个主机搭建，就配置几个主机ip地址

关闭防火墙

查看iptables和SElinux状态：

关闭iptables和SElinux：

永久关闭SElinux

vim /etc/selinux/config,然后将SELINUX改成disabled，这样即使重启操作系统SElinux也是关闭状态。

查看SElinux状态：

配置ssh免登陆

生成ssh免登陆密钥，进入到我的home目录下的.ssh文件夹

cd ~/.ssh

开始生成私钥和公钥

执行完这个命令后，会生成两个文件id_rsa（私钥）、id_rsa.pub（公钥）
将公钥拷贝到要免登陆的机器上

安装idk，配置环境变量

安装JDK

将 jdk-8u92-linux-x64.tar.gz上传到Linux 的 / 根目录下

解压jdk

将java添加到环境变量中

vim /etc/profile

#在文件最后添加
export JAVA_HOME=/usr/java/jdk1.8
export PATH=$PATH:$JAVA_HOME/bin

#刷新配置
source /etc/profile
